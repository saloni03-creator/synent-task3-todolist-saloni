# To-Do List (CLI)

## Project Description

The To-Do List is a Python command-line interface (CLI) project that helps users manage daily tasks efficiently. Users can add tasks, view existing tasks, and delete completed tasks using a simple menu-driven system.
This project is beginner-friendly and helps in understanding core Python concepts such as lists, functions, loops, conditional statements, and user input handling.

---

## Features

- Add new tasks
- View all tasks
- Delete completed tasks
- Menu-driven CLI interface
- Handles invalid inputs

---

## Technologies Used

- Python 3
- Command Line Interface (CLI)

---

## How to Run

### Step 1:
Install Python 3 on your system.

### Step 2:
Save the Python file as:
todo_list.py

Step 3:

Open terminal or PyCharm and run:
python todo_list.py
